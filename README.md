# Library Management System

A Java-based application designed to automate the daily operations of a library, such as managing books, tracking members, and handling borrowing/returning transactions.

## 🚀 Features
* **Inventory Management:** Add, update, and remove books from the system.
* **Member Tracking:** Register new members and manage their profiles.
* **Transaction System:** Record book issues and returns with date tracking.
* **Search:** Filter books by title, author, or book number.

## 🛠 Tech Stack
* **Language:** Java
* **Concepts:** Object-Oriented Programming (OOP), File I/O (or Database if applicable)

## 📝 How to Run
1. Clone the repository.
2. Open the project in your preferred Java IDE.
3. Run the `Main.java` file.
